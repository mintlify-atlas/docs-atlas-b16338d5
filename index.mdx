## Quick Start

Get up and running with Neovim from Scratch in just a few steps.

<Steps>
  <Step title="Check Prerequisites">
    Ensure you have Neovim v0.8.0 or higher installed. Learn more about [prerequisites](/prerequisites).
  </Step>
  <Step title="Install the Config">
    Clone the repository into your Neovim config directory and let the plugins install automatically.
    
    ```bash
    git clone https://github.com/LunarVim/Neovim-from-scratch.git ~/.config/nvim
    ```
  </Step>
  <Step title="Start Customizing">
    Explore the modular configuration structure and customize it to your needs. Check out our [customization guides](/customization/adding-plugins).
  </Step>
</Steps>
